# 💳 Digital Wallet Transaction Dashboard

An interactive **Digital Wallet Transaction Dashboard** developed using **Microsoft Excel** to analyze transaction performance, payment methods, transaction status, product categories, and geographic distribution. The dashboard provides key business insights through PivotTables, PivotCharts, slicers, and KPI cards, enabling users to explore transaction data efficiently.

---


## 🎯 Project Objectives

- Analyze digital wallet transaction performance.
- Monitor key business metrics using KPI cards.
- Compare payment method usage.
- Track transaction success and failure rates.
- Analyze transaction fee trends over time.
- Understand transaction distribution by location and product category.

## 📈 Dashboard Features

### 🔹 KPI Cards
Provides a quick overview of the dashboard's most important business metrics.

### 🔹 Product Category Analysis
Displays the total paid amount and transaction fees across different product categories.

### 🔹 Transaction Amount Analysis
Shows transaction counts and transaction fees grouped by transaction amount ranges.

### 🔹 Monthly Transaction Fee Trend
Visualizes monthly transaction fee trends to identify seasonal patterns and performance changes.

### 🔹 Payment Method Analysis
Compares transaction volumes across:
- Bank Transfer
- Credit Card
- Debit Card
- UPI
- Wallet Balance

### 🔹 Transaction Status Analysis
Displays the percentage of:
- Successful Transactions
- Failed Transactions
- Pending Transactions

### 🔹 Location Analysis
Shows the distribution of transactions across:
- Urban
- Suburban
- Rural

---

## 🎛 Interactive Filters

The dashboard includes slicers that allow users to filter data by:

- Year
- Payment Method
- Transaction Status

All charts and KPIs update dynamically based on the selected filters.

## 📊 Key Insights

- Approximately **95%** of transactions were successful.
- **Bank Transfer** is the most commonly used payment method.
- Urban areas account for the highest share of transactions.
- Transaction fees remain relatively stable across the selected period.
- Product categories contribute differently to total paid amounts and transaction fees, helping identify high-performing segments.

## 🚀 How to Use

1. Download or clone this repository.
2. Open the Excel workbook using **Microsoft Excel 2019, Excel 2021, or Microsoft 365**.
3. Enable editing if prompted.
4. Navigate to the **Dashboard** worksheet.
5. Use the slicers to interact with the dashboard and explore the data.
